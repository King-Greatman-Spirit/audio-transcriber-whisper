# 🎙️ Audio Transcriber (Whisper AI + Python)

Convert your long audio recordings into **text, Word (.docx), and PDF** using **OpenAI Whisper** – free, accurate, and unlimited.

---

## ✨ Features
- ✅ Transcribe audio (MP3, WAV, M4A, etc.) into **text**
- ✅ Save results as `.txt`, `.docx`, and `.pdf`
- ✅ Works offline (no API key required)
- ✅ Free & unlimited usage with Whisper AI

---

## 🚀 Quick Start

### 1️⃣ Clone the Repository
```bash
git clone https://github.com/King-Greatman-Spirit/audio-transcriber-whisper.git
cd audio-transcriber-whisper
````

### 2️⃣ Create Virtual Environment

```bash
python -m venv venv
venv\Scripts\activate   # Windows

```bash

```bash

## 🛠 Installing FFmpeg (Windows Guide)

Whisper **requires FFmpeg** for audio processing. Here’s how to set it up:

### Step 1: Download FFmpeg

1. Go to 👉 [https://www.gyan.dev/ffmpeg/builds/](https://www.gyan.dev/ffmpeg/builds/)
2. Download **ffmpeg-release-essentials.zip**

### Step 2: Extract FFmpeg

1. Extract the `.zip` into:

   ```
   C:\ffmpeg
   ```
2. You should now have:

   ```
   C:\ffmpeg\bin\ffmpeg.exe
   C:\ffmpeg\bin\ffplay.exe
   C:\ffmpeg\bin\ffprobe.exe
   ```

### Step 3: Add FFmpeg to PATH

1. Press **Win + R** → type `sysdm.cpl` → Enter
2. Go to **Advanced tab** → **Environment Variables**
3. Select `Path` under **System variables** → **Edit** → **New** → Add:

   ```
   C:\ffmpeg\bin
   ```
4. Save & close

### Step 4: Verify Installation

Open a new terminal and run:

```bash
ffmpeg -version
```

✅ You should see version details (e.g. `ffmpeg version n6.1`)
